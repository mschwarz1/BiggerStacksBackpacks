# BiggerStacksBackpacks

A Hytale content pack that adds stack multiplier upgrades to backpacks, allowing players to store more items per slot.

## Features

- **Backpack Capacity Upgrades** (Tiers 1-3): Overwrites vanilla backpack upgrades to use the StackMultiplierContainer system, adding 9 slots per tier while preserving stack multiplier support.
- **Stack Compressor Upgrades** (Tiers 1-6): New consumable items that double the backpack's stack multiplier each tier, up to 64x at max level. Tiered across Uncommon through Legendary quality with progressively harder crafting requirements.

## Requirements

- [StackMultiplierContainerLib](https://github.com/mschwarz1/StackMultiplierContainerLib) plugin installed on the server.

## Installation

1. Download the latest release ZIP.
2. Place the ZIP in your Hytale server's content packs directory.
3. Ensure StackMultiplierContainerLib is installed as a server plugin.
